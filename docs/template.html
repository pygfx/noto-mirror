Noto font mirror

This is a mirror of a subset of Noto fonts that together make up a default (fallback) font that supports nearly all scripts Unicode. The source code for this project is at https://github.com/pygfx/noto-mirror/.

You can download a font by clicking on it. Then open the download folder on your computer and double-click the font. This should open a dialog where you can press "install".
